# Project roadmap

This roadmap outlines about the needed works for this package to support and cover better functionality for rate limiting in an ASGI application.

## Short term (0.x.x releases):

- Add better documentation both for user, maintainer, or contributor
- Add more unit test for better coverage
- Bug fix if encountered
- Advance documentations and guide for advance scenarios
- FastAPI usage guide

## Mid term (0.1.x - 0.5.x releases):

- Move docs to a dedicated page
- Custom error handler
- Custom exception
- Plugable monitoring
- WSGI adapters
- Multiple python framework integration
- Multiple python framework guides

## Long term (+1.x.x releases):

- Stable API
- Matured contribution environment and workflow
- Automatic build and releases
